Simple Table: Array of Arrays

An array of arrays orderData structure can be easily converted in a TableModel with help of TableFactory .createTableModel().

Object Array

The most common use case is an array of business objects. This array can be easily converted to a table model. An array of ColumnDefIfs is used to define which object property is displayed in which table column.

Using the Table API

After the first rendering of a table, a TableApi service is emitted. With this service, you can do a lot of things: change cell values, change table column widths, scroll to a specific row, trigger a repaint, etc.
